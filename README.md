# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents a **Sales Performance Dashboard** built using Power BI.  
It provides insights into sales trends, profit analysis, and product performance to support data-driven decision making.

---

## 🎯 Objective
- Analyze sales trends over time  
- Identify top-performing products and categories  
- Evaluate profit and revenue distribution  
- Generate actionable business insights  

---

## 🛠️ Tools Used
- Power BI  
- Excel / CSV Dataset  
- Power Query (Data Cleaning)  
- DAX Measures  

---

## 📊 Dashboard Features

### 📈 Sales Analysis
- Sales trend over time  
- Monthly & category-wise performance  

### 🏆 Top Performers
- Best selling categories  
- Highest revenue products  

### 💰 Profit Analysis
- Profit trend  
- High vs low margin products  

### 📦 Product Insights
- Top vs underperforming products  
- Sales contribution analysis  

---

## 📌 Key Insights
- Sales show a consistent growth trend  
- Few products contribute to majority of revenue (80/20 rule)  
- Some categories generate high sales but low profit  
- Demand is concentrated on limited product groups  

---

## 🚀 Business Recommendations
- Focus on high-performing categories  
- Improve low-performing products  
- Maintain stock for top-selling items  
- Optimize pricing strategies
